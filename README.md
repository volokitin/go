# Денежный поток: мини-игра

Учебная консольная игра на Go, вдохновлённая идеей «денежного потока».
Цель — довести пассивный доход до 5000 в месяц, управляя доходами и расходами.

## Запуск

```bash
go run ./cmd/game
```

## Правила

- Каждый месяц получаете зарплату и пассивный доход.
- Оплачиваете обязательные расходы.
- Выбираете действие: инвестировать, учиться или копить.
- Случайные события могут помочь или навредить.
